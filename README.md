# Python Task Manager

## Description
This project is a command-line based Task Manager developed using Python.
It allows users to manage daily tasks efficiently.

## Features
- Add tasks with title, description, and due date
- View pending and completed tasks
- Mark tasks as completed
- Delete tasks by task ID
- Search tasks by keyword or due date
- Save and load tasks using JSON file

## How to Run
```bash
python task_manager.py
